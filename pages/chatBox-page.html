Chat Box App

About

The Chat Box App is an app I created as a performance task during my Senior High School days. We used MIT App Inventor to develop this app. The tool allows you to use code blocks and a drag-and-drop interface for designing the UI. While the tool is beginner-friendly and great for prototyping, it does have limitations in terms of functionality. Despite these constraints, it was a valuable learning experience that introduced me to the basics of app development and problem-solving. The Chat Box App enables users to chat with anyone, regardless of distance, as long as they have the app installed. It supports both one-to-one and many-to-many communication.
